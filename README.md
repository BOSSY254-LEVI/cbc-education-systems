# CBC / CBE National Education Platform

A scalable, secure, KICD-aligned digital platform for managing
Competency-Based Curriculum (CBC) and Competency-Based Education (CBE)
across schools, counties, and national education authorities.

## Features
- School login & dashboards
- Teacher and learner management
- Parent accounts linked to learners
- CBC-aligned assessments & reports
- Mobile apps for students and parents
- Secure, school-isolated data access

## Tech Stack
- Backend: NestJS (TypeScript)
- Web: React + Tailwind
- Mobile: React Native (Expo)
- Database/Auth: Supabase (PostgreSQL + RLS)
